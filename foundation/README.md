# KL-UNL Pipeline Foundations

## Foundation Files
Foundation files are designed to be referenced by an HTML file within the workspace for a simulation update.
  * A demonstration project (see demo_01.html) shows the reference protocols
  * Foundation files provide a uniform masthead and project style, and MathJax for displayed equations
    * Masthead:  kl-unl-masthead.js
    * Masthead:  contents.js
    * Style:     kl-unl.css
    * Equations: kl-unl.js
  * When creating a new simulation update, add an entry to the masthead contents file with modified text based on the original Help and About material
